The C programming language was developed in 1972 by Dennis Ritchie at Bell Labs. It was mainly created to develop the Unix operating system. Before C, programming was done in assembly language, which was difficult to use. C made programming easier, faster, and more efficient.

C99 (1999)
C11 (2011)
C17/C18 (2018)
C23 (2024)

Why C is Still Used Today:-
It is fast and efficient
It provides low-level memory access
It is used in embedded systems and microcontrollers
Many operating systems are written in C
It helps programmers understand computer fundamentals

Q5 Explain decision-making statements in C (if, else, nested if-else, switch).
Provide examples of each.

Decision-Making Statements in C Programming Language

Decision-making statements are used to control the flow of program based on conditions.

Types of Decision-Making Statements:

if
if-else
nested if-else
switch
1. if Statement

The if statement is used to execute code when condition is true.

Syntax
if(condition)
{
   statements;
}
Example
#include <stdio.h>
int main()
{
    int a = 10;

    if(a > 5)
    {
        printf("A is greater than 5");
    }

    return 0;
}
2. if-else Statement

The if-else statement executes one block if condition is true and another if false.

Syntax
if(condition)
{
   statements;
}
else
{
   statements;
}
Example
#include <stdio.h>
int main()
{
    int num = 7;

    if(num % 2 == 0)
    {
        printf("Even Number");
    }
    else
    {
        printf("Odd Number");
    }

    return 0;
}
3. Nested if-else

Nested if-else means if statement inside another if statement.

Example
#include <stdio.h>
int main()
{
    int a = 10, b = 20;

    if(a > b)
    {
        printf("A is greater");
    }
    else if(b > a)
    {
        printf("B is greater");
    }
    else
    {
        printf("Both are equal");
    }

    return 0;
}
4. switch Statement

The switch statement is used when there are multiple choices.

Syntax
switch(expression)
{
   case value1:
       statements;
       break;

   case value2:
       statements;
       break;

   default:
       statements;
}
Example
#include <stdio.h>
int main()
{
    int day = 2;

    switch(day)
    {
        case 1:
            printf("Monday");
            break;

        case 2:
            printf("Tuesday");
            break;

        case 3:
            printf("Wednesday");
            break;

        default:
            printf("Invalid Day");
    }

    return 0;
}

Q6 Compare and contrast while loops, for loops, and do-while loops. Explain the scenarios in which each loop is most appropriate.

Loops in C Programming Language 

Loops are used to repeat statements multiple times.

Types of loops:

while
for
do-while
1. while Loop

Checks condition before execution.

Syntax
while(condition)
{
   statements;
}
Example
int i = 1;
while(i <= 5)
{
   printf("%d", i);
   i++;
}

Use When:
Number of iterations is not known

2. for Loop

Used when number of iterations is known.

Syntax
for(initialization; condition; increment)
{
   statements;
}
Example
for(int i = 1; i <= 5; i++)
{
   printf("%d", i);
}

Use When:
Fixed number of times

3. do-while Loop

Executes at least once.

Syntax
do
{
   statements;
}
while(condition);
Example
int i = 1;
do
{
   printf("%d", i);
   i++;
}
while(i <= 5);

Use When:
Menu programs or input validation

Comparison
Loop	Condition	Runs
while	Before	0 or more times
for	Before	0 or more times
do-while	After	At least once

Q7 Explain the use of break, continue, and goto statements in C. Provide
examples of each.

Jump statements are used to control the flow of program.

Types:

break
continue
goto
1. break Statement

The break statement is used to exit from loop or switch immediately.

Example
#include <stdio.h>
int main()
{
    int i;

    for(i = 1; i <= 5; i++)
    {
        if(i == 3)
            break;

        printf("%d\n", i);
    }

    return 0;
}
Output
1
2

Use: Stop loop when condition is met

2. continue Statement

The continue statement skips current iteration and moves to next.

Example
#include <stdio.h>
int main()
{
    int i;

    for(i = 1; i <= 5; i++)
    {
        if(i == 3)
            continue;

        printf("%d\n", i);
    }

    return 0;
}
Output
1
2
4
5

Use: Skip specific condition

3. goto Statement

The goto statement transfers control to a labeled statement.

Example
#include <stdio.h>
int main()
{
    int i = 1;

start:
    printf("%d\n", i);
    i++;

    if(i <= 5)
        goto start;

    return 0;
}
Output
1
2
3
4
5

Q9 Explain the concept of arrays in C. Differentiate between one-dimensional and
multi-dimensional arrays with examples.

An array is a collection of same data type elements stored in continuous memory.

Example:

int a[5] = {10,20,30,40,50};
1. One-Dimensional Array

Stores elements in single row.

Example
#include <stdio.h>
int main()
{
   int a[5] = {1,2,3,4,5};
   for(int i=0;i<5;i++)
   {
      printf("%d ",a[i]);
   }
   return 0;
}
2. Multi-Dimensional Array

Stores elements in rows and columns.

Example
#include <stdio.h>
int main()
{
   int a[2][2] = {{1,2},{3,4}};
   for(int i=0;i<2;i++)
   {
      for(int j=0;j<2;j++)
      {
         printf("%d ",a[i][j]);
      }
      printf("\n");
   }
   return 0;
}
Difference
One-Dimensional	Multi-Dimensional
Single row	Rows and columns
One index	Two indexes
Example: a[5]	Example: a[2][2]

#include <string.h>

strcpy() is used to copy one string into another.

#include <stdio.h>
#include <string.h>

strcat() is used to join two strings.

Example
#include <stdio.h>
#include <string.h>

#include <stdio.h>

Q12 Explain the concept of structures in C. Describe how to declare, initialize, and
access structure members.

A structure is a user-defined data type used to store different types of data together.

Example: storing student details like name, age, and marks.

1. Structure Declaration

Structure is declared using struct keyword.

Syntax
struct structure_name
{
   data_type member1;
   data_type member2;
};
Example
struct student
{
   int roll;
   char name[20];
   float marks;
};
2. Structure Initialization

Structure variables are created after declaration.

Example
struct student s1 = {1, "Parth", 85.5};
3. Accessing Structure Members

Structure members are accessed using dot (.) operator

Example
printf("%d", s1.roll);
printf("%s", s1.name);
printf("%f", s1.marks);
Complete Example
#include <stdio.h>

struct student
{
   int roll;
   char name[20];
   float marks;
};

int main()
{
   struct student s1 = {1, "Parth", 85.5};

   printf("Roll = %d\n", s1.roll);
   printf("Name = %s\n", s1.name);
   printf("Marks = %.2f", s1.marks);

   return 0;
}
Output
Roll = 1
Name = Parth
Marks = 85.50

Q13  Explain the importance of file handling in C. Discuss how to perform file
operations like opening, closing, reading, and writing files

File handling is used to store data permanently in files instead of temporary memory.
It helps in saving, reading, and updating data.

Importance of File Handling
Store data permanently
Read data anytime
Handle large data
Useful for records (student, employee, etc.)
File Operations in C
Opening a File
Writing to File
Reading from File
Closing a File
1. Opening a File

Files are opened using fopen() function.

Syntax
FILE *fp;
fp = fopen("file.txt","mode");
Modes
Mode	Meaning
r	Read
w	Write
a	Append

Example

FILE *fp;
fp = fopen("data.txt","w");
2. Writing to File

Use fprintf() to write data.

Example
#include <stdio.h>

int main()
{
   FILE *fp;
   fp = fopen("data.txt","w");

   fprintf(fp,"Hello World");

   fclose(fp);
   return 0;
}
3. Reading from File

Use fscanf() to read data.

Example
#include <stdio.h>

int main()
{
   FILE *fp;
   char text[50];

   fp = fopen("data.txt","r");

   fscanf(fp,"%s",text);

   printf("%s",text);

   fclose(fp);
   return 0;
}
4. Closing a File

Use fclose() to close file.

fclose(fp);

Closing file is important to save data properly.

Summary
Function	Use
fopen()	Open file
fprintf()	Write file
fscanf()	Read file
fclose()	Close file
